# Impeccable project context

## Design Context

### Users
Anyone with a trivial daily decision and ten seconds to spare — phone in hand as
often as desktop. The job to be done: delegate a coin-flip, with more theatre
than a coin. Sessions last seconds; delight per second matters.

### Brand Personality
Silly, ceremonial, homespun. A loved late-90s web toy (5k-contest era) given a
proper poster treatment. It takes its absurd job completely seriously — that's
the joke.

### Aesthetic Direction
Sports-day-on-a-lawn: bright field green (from the original's #66CC33), chalk
lines, pixel bug sprites, a rubber-stamped verdict. Display type is Silkscreen
(period-authentic pixel face); body is a Verdana stack as homage to the original.
Single committed bright theme, no dark mode. Anti-references: SaaS dashboards,
cards, gradients, glassmorphism, anything that smells of a component library.

### Design Principles
1. The jerky, randomly-timed bug hop is the soul of the toy — never smooth it.
2. Retain the original's exact feature set; simplicity is the product.
3. One bold signature (the racetrack + stamp); everything around it stays quiet.
4. Period homage, modern floor: responsive to 320px, visible focus, reduced
   motion respected, verdict announced to screen readers.
